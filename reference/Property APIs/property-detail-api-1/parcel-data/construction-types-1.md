* "Frame"
* "Heavy Frame"
* "Steel Frame"
* "Wood Frame"
* "Metal Frames"
* "Light Frame"
* "Pole Frame"
* "Unknown"
* "Wood"
* "Masonry/Concrete Masonry Units (Cmus)"
* "Brick"
* "Stone"
* "Tilt-up Concrete"
* "Manufactured"
* "Mixed"
* "Log"
* "Steel & Concrete"
